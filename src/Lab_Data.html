Laboratory Data

This is the Laboratory Testing Data page. Jackson has no small amount of testing data, so I took the liberty of splitting it up into more digestable chunks based on usage, area, and/or testing. Please feel free to choose what you'd like to see.

  • Dust Suppression Data Mud Dam testing data compared against license limits for Dust Suppression.
  • Drill Muds for CCP Composite Drill Mud samples tested for use in Civil Construction Product.
  • Stormwater Data Stormwater testing against catch dams and landfill stormwater runoff.
  • Effluent and Irrigation Data Testing of effluent and irrigatable waste against irrigation criteria.
  • Bio-Remediation Test Data Testing of bio-remediation trial material.
  • Environmental Creek Data Environmental monitoring data for local creeks.
  • PFAS Data PFAS Testing data for solid and liquid wastes across site (IN DEVELOPMENT).
  • Pump Well Data PFAS Data for monitoring the Pump Well (IN DEVELOPMENT).

Currently, these are incomplete, but more progress is being made daily, so in time you will have all the data you want!
